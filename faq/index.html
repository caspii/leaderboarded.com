Getting Started

How do I create my first leaderboard? +

Creating a leaderboard is simple and takes less than 2 minutes:

  • Visit keepTheScore.com and click "Create Leaderboard"
  • Choose your leaderboard type (Competition, Points Tracker, or Tournament)
  • Add player names or enable self-registration
  • Customize colors, logos, and settings
  • Share the link with participants

No account is required to start, though creating one lets you save and manage multiple leaderboards.

Do I need technical skills to use keepTheScore? +

No technical skills are required! keepTheScore is designed to be user-friendly for everyone. The interface is intuitive, and you can create professional-looking leaderboards without any coding or design experience.

We also provide templates and guides to help you get started quickly.

Can multiple people update the same leaderboard? +

Yes! You have several options for collaborative leaderboards:

  • Admin access: Share admin links with trusted team members
  • Player self-scoring: Allow participants to update their own scores
  • Moderator roles: Assign specific permissions to different users
  • Public submissions: Enable score submissions that you can approve

All updates happen in real-time, so everyone sees changes immediately.

Is keepTheScore free to use? +

Yes! keepTheScore offers a generous free plan that includes:

  • Unlimited leaderboards
  • Up to 50 players per leaderboard
  • Real-time updates
  • Basic customization options
  • Mobile-friendly display

Premium plans are available for advanced features like custom branding, more players, API access, and priority support.

Features & Customization

Can I customize the appearance of my leaderboard? +

Absolutely! keepTheScore offers extensive customization options:

  • Colors: Choose your brand colors or use pre-made themes
  • Logos: Upload your company or team logo
  • Backgrounds: Add custom background images or patterns
  • Fonts: Select from various font styles
  • Layout: Choose between different display formats
  • Player images: Add profile pictures for each participant

Does the leaderboard update in real-time? +

Yes! All leaderboards update automatically in real-time. When someone's score changes, everyone viewing the leaderboard sees the update immediately without needing to refresh the page.

This real-time functionality works across all devices - desktop, tablet, and mobile - ensuring everyone stays synchronized.

Can I embed the leaderboard on my website? +

Yes, you can easily embed your leaderboard on any website using our embed code:

  • Copy the provided iframe code from your leaderboard settings
  • Paste it into your website's HTML
  • Adjust the width and height as needed
  • The embedded leaderboard updates in real-time just like the original

The embed feature works with all website builders including WordPress, Wix, Squarespace, and custom HTML sites.

What types of competitions can I track? +

keepTheScore is versatile and can track virtually any type of competition:

  • Sports: Tournaments, league tables, fitness challenges
  • Sales: Performance tracking, targets, commissions
  • Education: Class points, reading challenges, behavior tracking
  • Gaming: Esports tournaments, high scores, achievements
  • Fundraising: Donation tracking, campaign progress
  • Custom: Any activity where you need to track and compare scores

Technical & Security

Is my data secure? +

Yes, we take data security seriously:

  • All data is encrypted in transit and at rest
  • We use secure HTTPS connections
  • Regular security audits and updates
  • GDPR compliant data handling
  • You can delete your data at any time
  • No sharing of data with third parties

We also offer private leaderboards that require passwords for viewing.

Can I export my leaderboard data? +

Yes! You can export your data at any time:

  • CSV format: For use in Excel or Google Sheets
  • JSON format: For developers and integrations
  • PDF reports: For printing or sharing
  • Image snapshots: For social media or presentations

Premium plans also include scheduled automatic exports and API access for real-time data integration.

Does it work on mobile devices? +

Yes! keepTheScore is fully responsive and works perfectly on all devices:

  • Smartphones (iOS and Android)
  • Tablets and iPads
  • Desktop computers
  • Smart TVs and displays

The interface automatically adjusts to provide the best viewing experience on each device size. You can also use our TV mode for large display screens.
